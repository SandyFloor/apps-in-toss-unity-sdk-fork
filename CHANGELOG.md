# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-08-29

### Added
- 최초 Apps in Toss Unity SDK 릴리스
- Unity 프로젝트를 Apps in Toss 미니앱으로 변환하는 기본 기능
- 토스페이 결제 시스템 연동
- 광고 시스템 (배너, 전면, 보상형 광고)
- 사용자 인증 및 로그인 시스템
- 플랫폼 API 연동 (저장소, 공유, 알림 등)
- WebGL 템플릿 최적화
- 성능 모니터링 도구
- 자동 빌드 및 배포 도구
- Unity 2019.4 LTS 이상 지원
- Tuanjie Engine 지원

### Core Features
- **Platform Integration**: 완전한 Apps in Toss 플랫폼 연동
- **Payment System**: 토스페이 간편 결제 및 인앱 결제
- **Advertisement**: 다양한 형태의 광고 시스템
- **User Authentication**: 토스 계정 기반 로그인
- **Analytics**: 게임 분석 및 성능 모니터링
- **Optimization**: 모바일 환경 최적화

### Developer Tools
- **Editor Window**: Unity Editor 내 설정 및 빌드 도구
- **Auto Conversion**: 자동 코드 변환 및 최적화
- **Debug Tools**: 실시간 디버깅 및 로그 시스템
- **Performance Profiler**: 성능 프로파일링 도구

### Supported APIs
- Payment API (토스페이, 인앱 결제)
- Advertisement API (배너, 전면, 보상형)
- User API (인증, 정보 조회)
- Platform API (저장소, 공유, 알림)
- Analytics API (이벤트 추적, 통계)
- Device API (기기 정보, 권한)

### Known Issues
- 일부 Unity 2019.x 버전에서 빌드 경고 발생 (기능에는 영향 없음)
- WebGL 빌드 시 텍스처 압축 최적화 필요

### Documentation
- 완전한 API 문서 제공
- 샘플 프로젝트 및 튜토리얼
- 개발자 가이드 및 FAQ
